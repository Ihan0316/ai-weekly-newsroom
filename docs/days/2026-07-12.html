#운영체제

운영체제에서 메모리 관리 기법 중, 가상 메모리를 구현하기 위해 물리 메모리를 고정된 크기의 블록으로 나누고, 프로세스의 논리 주소 공간도 같은 크기로 나누어 관리하는 기법은?

  1. ①세그먼테이션(Segmentation)
  2. ②페이징(Paging)
  3. ③스와핑(Swapping)
  4. ④단편화(Fragmentation)

보기를 클릭해 정답을 확인하세요.

정답: ②

페이징은 가상 메모리를 구현하기 위한 메모리 관리 기법입니다. 물리 메모리와 논리 메모리 모두 고정된 크기의 블록(페이지와 프레임)으로 나누어 관리하며, 외부 단편화 문제를 해결하는 데 효과적입니다. 세그먼테이션은 가변 크기의 논리적 단위로 메모리를 관리하는 방식입니다.

03
IT · 개발 · 기획 용어
  • 개발 컨테이너 오케스트레이션 (Container Orchestration) 여러 컨테이너화된 애플리케이션의 배포, 관리, 확장 및 네트워킹을 자동화하는 기술입니다.
  • 개발 백프레셔 (Backpressure) 데이터 처리 시스템에서 소비자(Consumer)가 생산자(Producer)의 데이터 생성 속도를 제어하여 과부하를 방지하는 메커니즘입니다.
  • 기획 그릿 (Grit) 장기적인 목표를 향해 열정과 끈기를 가지고 노력하는 역량을 의미하며, 기획 과정에서 문제 해결과 목표 달성에 중요한 요소입니다.
