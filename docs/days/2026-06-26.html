정답: ②

테이블은 관계형 데이터베이스에서 데이터를 실제로 저장하는 기본 구조입니다. 행은 레코드(튜플)를, 열은 속성(애트리뷰트)을 나타냅니다. 뷰는 하나 이상의 테이블에서 파생된 가상 테이블이며, 인덱스는 데이터 검색 속도를 높이는 자료구조이고, 스키마는 데이터베이스의 전체적인 구조와 제약 조건을 정의합니다.

  • IT 로드밸런싱 (Load Balancing) 네트워크 트래픽을 여러 서버에 분산하여 특정 서버에 부하가 집중되는 것을 방지하고 서비스의 안정성과 성능을 향상시키는 기술입니다.
