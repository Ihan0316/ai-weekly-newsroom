#운영체제

운영체제에서 여러 프로세스가 공유 자원에 접근할 때, 한 시점에 하나의 프로세스만 접근하도록 제어하여 데이터 일관성을 유지하는 기법은?

  1. ①인터럽트(Interrupt)
  2. ②스레싱(Thrashing)
  3. ③뮤텍스(Mutex)
  4. ④페이징(Paging)

보기를 클릭해 정답을 확인하세요.

정답: ③

뮤텍스(Mutex)는 상호 배제(Mutual Exclusion)를 의미하며, 공유 자원에 대한 접근을 제어하여 한 번에 하나의 프로세스 또는 스레드만 접근하도록 보장합니다. 이를 통해 여러 프로세스나 스레드가 동시에 공유 자원을 변경하여 발생하는 데이터 불일치 문제를 방지하고 데이터 일관성을 유지할 수 있습니다.

IT · 개발 · 기획 용어
  • IT 데이터 레이크하우스 (Data Lakehouse) 데이터 레이크의 유연성과 데이터 웨어하우스의 구조 및 관리 기능을 결합하여, 대규모 데이터의 저장, 처리, 분석을 효율적으로 수행하는 아키텍처입니다.
  • 개발 도메인 특화 언어 (Domain-Specific Language, DSL) 특정 도메인이나 문제 영역에 최적화되어 해당 분야의 문제를 효율적으로 표현하고 해결할 수 있도록 설계된 프로그래밍 언어입니다.
  • 기획 그로스 루프 (Growth Loop) 제품이나 서비스의 성장을 지속적으로 촉진하기 위해 사용자가 제품을 통해 가치를 얻고, 그 가치를 통해 다시 새로운 사용자를 유입하거나 기존 사용자의 참여를 증대시키는 선순환 구조입니다.
