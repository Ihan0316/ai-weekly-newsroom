#운영체제

운영체제에서 메모리 관리 기법 중, 외부 단편화(External Fragmentation) 문제를 해결하기 위해 주기억장치에 연속적으로 할당된 프로세스의 메모리 공간을 재배치하는 기법은?

  1. ①페이징(Paging)
  2. ②세그멘테이션(Segmentation)
  3. ③압축(Compaction)
  4. ④스와핑(Swapping)

보기를 클릭해 정답을 확인하세요.

정답: ③

압축(Compaction)은 외부 단편화로 인해 생긴 분산된 빈 공간들을 모아 하나의 큰 연속된 공간으로 만드는 기법입니다. 이를 통해 더 큰 프로세스를 메모리에 적재할 수 있게 됩니다. 페이징과 세그멘테이션은 각각 고정/가변 크기로 메모리를 관리하여 단편화 문제를 완화하지만, 압축처럼 직접 공간을 재배치하지는 않습니다. 스와핑은 프로세스 전체를 메모리와 보조 기억장치 간에 교환하는 기법입니다.
