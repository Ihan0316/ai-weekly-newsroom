#운영체제

운영체제에서 메모리 관리 기법 중, 가상 메모리(Virtual Memory)의 구현을 위해 물리 메모리를 고정된 크기의 블록으로 나누고, 프로세스의 논리 주소 공간도 같은 크기로 나누어 관리하는 기법은?

  1. ①세그멘테이션(Segmentation)
  2. ②페이징(Paging)
  3. ③스와핑(Swapping)
  4. ④블록 매핑(Block Mapping)

보기를 클릭해 정답을 확인하세요.

정답: ②

페이징은 가상 메모리 구현을 위해 물리 메모리를 페이지 프레임이라는 고정 크기 블록으로, 프로세스의 논리 주소 공간을 페이지라는 같은 크기 블록으로 나누어 관리합니다. 이를 통해 외부 단편화를 해결하고 효율적인 메모리 관리가 가능해집니다.

IT · 개발 · 기획 용어
  • 개발 컨테이너 오케스트레이션 (Container Orchestration) 수많은 컨테이너화된 애플리케이션의 배포, 관리, 스케일링 및 네트워킹을 자동화하는 기술을 의미합니다.
  • IT 도메인 (Domain) 인터넷에서 컴퓨터 네트워크나 웹사이트의 주소를 나타내는 고유한 이름을 의미하며, IP 주소를 사람이 읽기 쉬운 형태로 변환한 것입니다.
  • 기획 온보딩 (Onboarding) 사용자가 새로운 제품이나 서비스를 처음 접했을 때, 쉽고 효과적으로 사용법을 익히고 가치를 경험하도록 돕는 일련의 과정을 말합니다.
