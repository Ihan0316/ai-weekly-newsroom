선점형(Preemptive) CPU 스케줄링 기법에 해당하는 것은?

  1. ①FCFS
  2. ②SJF(비선점)
  3. ③라운드 로빈(Round Robin)
  4. ④HRN

보기를 클릭해 정답을 확인하세요.

정답: ③

라운드 로빈은 각 프로세스에 동일한 타임 슬라이스(시간 할당량)를 주고 만료 시 강제로 CPU를 회수하는 선점형 기법이다. FCFS, 비선점 SJF, HRN은 실행 중인 프로세스를 도중에 뺏지 않는 비선점형 스케줄링이다.
