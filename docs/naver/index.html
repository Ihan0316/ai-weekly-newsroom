🧠 오늘의 퀴즈 (정보처리기사)
━━━━━━━━━━━━━━━━━━━━
[소프트웨어 공학] 소프트웨어 아키텍처 패턴 중, 여러 계층으로 분리하여 각 계층이 특정 역할만 수행하도록 하는 방식으로, 유지보수성과 확장성을 높이는 데 유리한 패턴은?
① 클라이언트-서버 패턴
② 파이프-필터 패턴
③ 계층형 패턴
④ 브로커 패턴

정답: 3번
해설: 계층형 패턴(Layered Pattern)은 시스템을 여러 계층으로 분리하여 각 계층이 고유한 역할과 책임을 가지도록 하는 아키텍처 패턴입니다. 이를 통해 각 계층의 독립성을 보장하고, 변경이 발생해도 다른 계층에 미치는 영향을 최소화하여 유지보수성과 확장성을 향상시킵니다.

📖 오늘의 IT 용어
━━━━━━━━━━━━━━━━━━━━

• 플랫폼 엔지니어링 (IT)
  개발팀이 소프트웨어를 더 쉽고 빠르게 개발, 배포, 운영할 수 있도록 셀프서비스 기능을 갖춘 내부 개발자 플랫폼을 구축하고 관리하는 분야입니다.

• 옵저버빌리티 (Observability) (개발)
  시스템의 외부 출력(로그, 메트릭, 트레이스 등)을 통해 시스템의 내부 상태를 얼마나 잘 추론하고 이해할 수 있는지를 나타내는 척도입니다.

• GTM (Go-To-Market) (기획)
  신제품이나 서비스를 시장에 성공적으로 출시하기 위한 전략적인 계획과 실행 과정을 의미합니다.
